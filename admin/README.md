# لوحة تحكم افراحنا- Admin Dashboard

لوحة تحكم منفصلة لإدارة تطبيق فرح.

## المميزات

- ✅ تسجيل دخول آمن للمدراء
- ✅ إحصائيات شاملة للنظام
- ✅ إدارة المستخدمين (عرض، تعديل، حذف)
- ✅ إدارة القاعات (عرض، تفعيل/تعطيل، حذف)
- ✅ إدارة الخدمات (عرض، تفعيل/تعطيل، حذف)
- ✅ إدارة الحجوزات (عرض، تحديث الحالة)
- ✅ نظام صلاحيات متقدم
- ✅ واجهة مستخدم عربية حديثة

## الوصول

1. تأكد من تشغيل الـ backend server
2. افتح المتصفح على: `http://localhost:5000/admin/login.html`
3. سجل الدخول بحساب admin

## متطلبات

- حساب مستخدم برتبة `ADMIN`
- كلمة مرور صحيحة

## الأمان

- جميع الطلبات تتطلب JWT token
- التحقق من صلاحيات ADMIN قبل الوصول
- تسجيل خروج تلقائي عند انتهاء الجلسة

## الملفات

- `login.html` - صفحة تسجيل الدخول
- `dashboard.html` - لوحة التحكم الرئيسية
- `index.html` - (ملف قديم - يمكن حذفه)

## API Endpoints المستخدمة

- `POST /api/auth/login` - تسجيل الدخول
- `GET /api/admin/stats` - الإحصائيات
- `GET /api/admin/users` - قائمة المستخدمين
- `GET /api/admin/venues` - قائمة القاعات
- `GET /api/admin/services` - قائمة الخدمات
- `GET /api/admin/bookings` - قائمة الحجوزات
- `PATCH /api/admin/*/:id/status` - تحديث الحالة
- `DELETE /api/admin/*/:id` - حذف عنصر

## ملاحظات

- تأكد من أن الـ backend يعمل على المنفذ 5000
- في حالة تغيير المنفذ، قم بتحديث `API_BASE_URL` في الملفات
- جميع الطلبات تحتاج إلى Bearer token في header
